Scripting
=========

This section covers a various aspectecs related to code your game in Thunder Engine.

Thunder Engine supports a number of programming languages to write your game logic.
Each programing language wrapped into separate module and you are able to include or exclude any.

The main engine language is C++ and most of embedded game objects is written on this programming language.
The architecture of game engine allows to developer mix these modules.
However it's preferebale to use only one scripting language in your project.
